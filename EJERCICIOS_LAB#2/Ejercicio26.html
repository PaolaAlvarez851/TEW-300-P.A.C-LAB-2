Texto con Todas las Etiquetas y Estructura Completa

Este es un ejemplo que combina varias etiquetas de texto: Texto en negrita, Texto en cursiva, Texto resaltado, Texto eliminado, Texto insertado, H2O (subíndice), E = mc2 (superíndice).

La combinación de todas estas etiquetas mejora la presentación y estructura del contenido. HTML es el lenguaje base para el desarrollo web.
